# Basic workflow 

* Clone repository
* Create a new branch and switch to it
* Add files
* Commit
* Merge branches
* Push 

```sh
# clone
git clone <url>

# move into directory
cd <directory name>

# create a file
echo 'hello world' > new-file.txt

# add file
git add new-file.txt

# commit message
git commit -m 'adding a new file with sample text'

# push
git push origin master
```
